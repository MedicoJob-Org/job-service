# Job Service

Manages job listings, applications, and real-time notifications via Socket.io.

## 🚀 Features
- **Containerized**: Built with multi-stage Dockerfiles.
- **DevSecOps Ready**: Automated SAST and Container Scanning.
- **Kubernetes Orchestration**: Deployment manifests included in `/k8s`.

## 🛠️ Local Development
1. Install dependencies: `npm install`
2. Start the service: `npm start`
- Service runs on port: **5002**

## 🛡️ Security
This repository is monitored by **CodeQL** and **Trivy** via GitHub Actions.
